# Data Directory

- `defi_backtest.db` stores the historical prices, gas data, strategies, and saved backtest records.
- `signal.jpg` is the original logo asset used by the welcome page.
- `signal_sidebar.png` is the gray-background logo generated for the sidebar.
- `python prepare_logo.py` regenerates `signal_sidebar.png` from `signal.jpg`.
- This folder is reserved for reproducible sample data and exported experiment results.
